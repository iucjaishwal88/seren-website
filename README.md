# 🕯️ Seren — Light Your Calm

A premium handcrafted candle brand website built entirely using AI tools — zero coding required.

🌐 **Live Website:** [seren-candles.netlify.app](https://seren-candles.netlify.app)

---

## ✨ About This Project

This is a real, live website built for **Seren** — a handcrafted candle and gift hamper brand based in Muzaffarpur, Bihar, India.

Built by **Ayushi** — an aspiring AI entrepreneur and student — using only AI tools and prompts. No coding experience required.

- Handmade floral candle bouquets and gift hampers
- Products include rose, tulip, sunflower and teddy bear shaped candles
